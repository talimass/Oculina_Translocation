## *O. patagonica* genome annotation

Bash scripts were used to produce protein annotation using eggnog 5 and interproscan 5 databases. The databases were combined into an annotation table (see [results](https://github.com/talimass/Oculina_Translocation/tree/main/Transcriptomics/Genome_annotation/results)) with [joinanno.R](joinanno.R) script.
